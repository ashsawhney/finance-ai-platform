# Finance AI Platform

A deployable Next.js starter for a personal financial management and recommendation product.

## What is included
- Premium landing page
- Mock dashboard UI
- Pricing page
- Contact/demo request form API route
- Clean styling with no external UI dependencies
- Ready for GitHub + Vercel deployment

## Run locally
```bash
npm install
npm run dev
```

## Deploy to GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/finance-ai-platform.git
git push -u origin main
```

## Deploy to Vercel
1. Import the GitHub repo into Vercel
2. Framework preset: Next.js
3. Deploy

## Notes
This version is frontend-first and uses mock data so you can get the product live quickly. After deployment, the next step is wiring real auth, storage, AI extraction, and financial logic.
