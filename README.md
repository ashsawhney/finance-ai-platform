# Finance AI Personal

A simple personal finance dashboard starter built with Next.js and designed to deploy cleanly on Vercel.

## What is included
- Landing page
- Simple dashboard
- Upload page
- Manual profile page
- Clean CSS with no extra UI libraries

## Run locally
1. Install Node.js LTS
2. Open a terminal in this folder
3. Run:

```bash
npm install
npm run dev
```

4. Open `http://localhost:3000`

## Push to GitHub
Upload the full contents of this folder to your GitHub repo.

## Deploy to Vercel
1. Open Vercel
2. Import your GitHub repo
3. Deploy

No environment variables are required for this simple version.

## Notes
This is a frontend starter for personal use. It does not yet include:
- database
- authentication
- AI processing
- real financial integrations

Those can be added later once the basic site is live.
